# AI Tools Affiliate Site

Live website: https://aitools.lovehappyhours.com

## 🚀 Quick Start

This is a static website for reviewing AI tools and earning affiliate commissions.

## 📁 Project Structure

```
aitools-affiliate/
├── index.html              # Main website page
├── content/               # Review content
│   ├── reviews/          # Individual tool reviews
│   ├── guides/           # How-to guides
│   └── comparisons/      # Tool comparisons
├── assets/               # Images, CSS, JS
├── scripts/              # Automation scripts
└── data/                 # Analytics, tracking data
```

## 🎯 Features

- **Responsive Design**: Works on mobile & desktop
- **Affiliate Links**: Ready for commission tracking
- **SEO Optimized**: Basic SEO structure in place
- **Easy to Update**: Simple HTML/CSS

## 💰 Monetization

1. **Affiliate Commissions**: 30-50% on AI tool sales

## 🚀 Deployment Options

### Option 1: Netlify (Recommended)
```bash
# Deploy to Netlify
netlify deploy --prod --dir=.
```

### Option 2: GitHub Pages
1. Push to GitHub
2. Enable GitHub Pages in settings
3. Set custom domain

### Option 3: Any Static Hosting
Upload all files to any web host

## 📊 Analytics Setup

1. **Google Analytics**: Add tracking code to index.html
2. **Affiliate Tracking**: Use program dashboards
3. **Conversion Tracking**: Set up goal tracking

## 🔧 Development

```bash
# Local preview
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## 📝 Content Management

Add new reviews to `content/reviews/` directory. Each review should include:
- Tool overview
- Features list
- Pros & cons
- Pricing comparison
- Affiliate link
- Call to action

## 🎨 Customization

Edit `index.html` to:
- Change colors in CSS
- Add new tool cards
- Update pricing
- Modify layout
