🚀 Quick Setup Guide

1 Get Google Analytics ID

Time needed: 10 minutes

Steps:

  1. Go to analytics.google.com
  2. Click "Start measuring"
  3. Create account: "AI Tools Affiliate"
  4. Set up property: "aitools.lovehappyhours.com"
  5. Get Measurement ID (starts with G-)
2 Enable Tracking

Replace G-XXXXXXXXXX in the code above with your actual Measurement ID.

Files that need tracking:

  • index.html - Main landing page
  • analytics.html - This dashboard
  • All review pages in content/reviews/
3 Track Affiliate Clicks

Add this code after the main gtag config to track affiliate clicks:

// Track affiliate link clicks document.addEventListener('click', function(e) { if (e.target.matches('a[href*="affiliate"]') || e.target.closest('a[href*="affiliate"]')) { gtag('event', 'affiliate_click', { 'event_category': 'affiliate', 'event_label': e.target.href }); } });
