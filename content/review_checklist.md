# 📋 AI Tool Review Checklist

Use this checklist for each review you write.

## ✅ Before Writing
- [ ] **Choose tool** to review
- [ ] **Test the tool** (if possible)
- [ ] **Research** features and pricing
- [ ] **Check** for affiliate program
- [ ] **Gather** screenshots/images

## ✅ Content Structure
- [ ] **Title**: [Tool Name] Review 2026
- [ ] **Quick summary** (3-5 bullet points)
- [ ] **Key features** (5-10 points)
- [ ] **Pros & cons** (balanced view)
- [ ] **Pricing table** (all plans)
- [ ] **Comparison** to alternatives
- [ ] **Getting started** guide
- [ ] **Tips** for best results
- [ ] **Final verdict** with rating
- [ ] **Affiliate link** placement

## ✅ Quality Checks
- [ ] **Word count**: 1500+ words
- [ ] **Readability**: Clear and engaging
- [ ] **Accuracy**: Facts are correct
- [ ] **Uniqueness**: Original content
- [ ] **SEO**: Keywords included naturally
- [ ] **Images**: Screenshots added
- [ ] **Links**: Affiliate links working
- [ ] **Disclosure**: Affiliate disclaimer

## ✅ Publishing
- [ ] **Add to website** structure
- [ ] **Update** main page links
- [ ] **Test** on mobile/desktop
- [ ] **Share** on social media
- [ ] **Track** in analytics

## ⏰ Time Management
- **Research**: 1-2 hours
- **Writing**: 2-3 hours  
- **Editing**: 1 hour
- **Publishing**: 30 minutes
- **Total**: 4-6 hours per review

## 🎯 Weekly Goals
- **Week 1**: 2 reviews (ChatGPT + one other)
- **Week 2**: 3 reviews (total 5)
- **Week 3**: 2 reviews (total 7)
- **Week 4**: 3 reviews (total 10)

## 💰 Revenue Targets
- **Month 1**: $100-500 (2-5 reviews)
- **Month 2**: $500-1000 (5-10 reviews)
- **Month 3**: $1000-2000 (10-15 reviews)

## 🔧 Tools to Help
- **Grammarly**: Grammar checking
- **Hemingway**: Readability
- **Canva**: Image creation
- **Google Docs**: Collaboration
- **Trello**: Task management
